# **P03: Kanban del Projecte**

## **Breu Descripció**

* La **metodologia Kanban** és essencial per al seguiment i l’organització del projecte.
* A partir del **product backlog proporcionat**, cal gestionar i controlar les tasques de manera visual i ordenada.

## **Objectius**

1. **Seguiment de tasques**

   * Moure les tasques pel Kanban segons l’estat: *Per fer / En procés / Fet*.
   * Identificar quines tasques estan bloquejades o necessiten suport.

2. **Planificació setmanal**

   * Cada setmana es farà una **revisió de l’estat del Planner**.
   * Detectar desviacions i ajustar prioritzacions.

3. **Gestió del Product Backlog**

   * Assignar responsabilitats i deadlines.
   * Mantenir una visió clara de l’estat global del projecte.

## **Bones pràctiques**

* Actualitzar el Kanban **regularment**, no deixar-ho per última hora.
* Fer servir etiquetes o colors per identificar tipus de tasques o urgències.
* Documentar incidències o dependències que afectin l’avanç de les tasques.
